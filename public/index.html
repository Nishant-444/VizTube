Welcome to the VizTube API

The backend service for a modern video-sharing platform.

Live API Endpoint

https://viztube.onrender.com

How to Use This API

This is a backend-only project. To interact with the API, you can use the comprehensive Postman collection provided in the GitHub repository. It contains pre-configured requests for every endpoint.

View on GitHub

Example: Health Check

You can quickly check if the API is live by making a `GET` request to the root endpoint using any HTTP client or directly in your browser's console.

JavaScript Fetch Example 
fetch('https://viztube.onrender.com')
    .then(response => response.json())
    .then(data => console.log(data));
